# flex-odbc C++ Sample Driver

This is a sample implementation of an ODBC driver using the flex-odbc SDK, demonstrating how to create a C++ driver that interacts with the flex-odbc s library.

## Project Structure
The sample is located in `/samples/cpp_sample` and includes:
- CMakeLists.txt: Cross-platform build configuration
- main.cpp: Sample driver implementation
- CMakeLists.txt: Platform-specific configurations


This is a sample C++ ODBC driver implementation using the flex-odbc SDK.

## Getting Started
## Setup
1. Clone the flex-odbc repository
2. Install required dependencies:
   - CMake 3.15+
   - ODBC driver manager (e.g., unixODBC or Windows ODBC)
3. Navigate to the cpp_sample directory
4. Run cmake and build:
   ```bash
   mkdir -p build && cd build
   cmake ..
   make
   ```
5. The compiled executable will be in the build directory
